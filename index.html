Algoritmen

Op deze site kun je gebruik maken van drie verschillende soorten algoritmen.

Namelijk: linear search, binary search en bubble sort.

Linear search is een methode om een element in een lijst te vinden. Het controleert achtereenvolgens elk element van de lijst totdat er een overeenkomst is gevonden of de hele lijst is doorzocht.

Binary search vindt de positie van een doelwaarde binnen een gesorteerde array. Bij binair zoeken wordt de doelwaarde vergeleken met het middelste element van de array.

Bubble sort stapt herhaaldelijk element voor element door de invoerlijst, vergelijkt het huidige element met het element erna, en verwisselt indien nodig hun waarden.
